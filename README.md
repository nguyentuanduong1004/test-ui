# test-ui

This is a blank Angular 19 application scaffolded with standalone components and SCSS styling.

## Scripts

- `npm start` or `ng serve`: Serve the application locally
- `npm run build` or `ng build`: Build the application
- `npm run lint` or `ng lint`: Lint the codebase
- `npm run format` or `prettier --write .`: Format code

## Structure

- src/
  - app/
    - app.component.ts (standalone component)
    - app.component.html
    - app.component.scss
  - assets/ (static assets)
  - styles.scss (global styles)
  - main.ts (bootstrap)
  - polyfills.ts (zone.js)
  - index.html (entry HTML)
